App Deployment

The LR FAQ App is deployed on two Cloud environments: Pivotal Web Services © and MongoLab © MongoDB PaaS Cloud. A RESTful HTTP binding has been established between these Cloud based PaaS environments

Luckily, both of the Cloud PaaS environments are controlled from a single web account
